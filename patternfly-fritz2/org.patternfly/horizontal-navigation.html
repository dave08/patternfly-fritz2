fun <T> RenderContext.horizontalNavigation(router: Router<T>, store: NavigationStore<T> = NavigationStore(), id: String? = null, baseClass: String? = null, content: Navigation<T>.() -> Unit = {}): Navigation<T>

Creates a horizontal Navigation component.

Samples

import dev.fritz2.dom.html.render
import dev.fritz2.routing.router
import org.patternfly.group
import org.patternfly.groups
import org.patternfly.horizontalNavigation
import org.patternfly.item
import org.patternfly.items
import org.patternfly.tertiaryNavigation
import org.patternfly.verticalNavigation
fun main() { 
   //sampleStart 
   render {
    val router = router("home")
    horizontalNavigation(router) {
        items {
            item("get-started", "Get Started")
            item("get-in-touch", "Get in Touch")
        }
    }
} 
   //sampleEnd
}

Parameters

router

the router instance

store

the store for the navigation items

id

the ID of the element

baseClass

optional CSS class that should be applied to the element

content

a lambda expression for setting up the component itself
