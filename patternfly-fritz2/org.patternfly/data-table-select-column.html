Creates a SelectColumn to select rows in the data table. Use this function once to add a checkbox or radio button depending on the value of selectionMode.

Parameters

selectionMode

controls how rows can be selected

id

the ID of the cell element

baseClass

optional CSS class that should be applied to the cell element
